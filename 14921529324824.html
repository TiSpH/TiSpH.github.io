Arch Linux快速安装 (适用于大部分现代电脑)

2017/4/14 posted in  Tips

基本安装

  1. 分区, 1M 的 BIOS boot (sda1) 和剩余主分区 (sda2) cfdisk
  2. 将主分区格式化为 ext4 mkfs.ext4 /dev/sda2
  3. 挂载主分区 mount /dev/sda2 /mnt
  4. 选择镜像 nano /etc/pacman.d/mirrorlist
  5. 安装 pacstrap /mnt
  6. 更改 root 至新系统 arch-chroot /mnt /bin/bash
  7. 生成 fstab 文件 genfstab -U /mnt >> /mnt/etc/fstab
  8. 下载 grub pacman -S grub-bios
  9. 安装 grub grub-install /dev/sda grub-mkconfig -o /boot/grub/grub.cfg
  10. 启用 dhcpcd systemctl enable dhcpcd
  11. 重启, 完成剩余个性化设置

图形界面

  1. 安装通用显卡驱动 (性能不如专用) pacman -S xf86-video-vesa
  2. 安装触摸板驱动* pacman -S xf86-input-synaptics
  3. 安装 xorg pacman -S xorg-server xorg-xinit
  4. 安装 KDE 或者其他桌面环境 pacman -S plasma
  5. 在.xinitrc中添加exec startkde
  6. startx或者安装显示管理器
