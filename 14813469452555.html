Raspberry Pi Settings

Hardware:

  • Raspberry Pi 2 or later
  • Raspberry Pi Touch Display or HDMI Display
  • 5V 2.5A DC adapter
  • 16GB C10 microSD card (TF card)
  • USB wireless network card (Raspberry Pi 3 has built-in wireless LAN) or networking cables
  • Keyboard & mouse

System installation:

  • Use noobs or install by self.

Display Configuration

sudo nano /boot/config.txt

# Improve VNC Resolution:
hdmi_ignore_edid=0xa5000080
hdmi_group=2
hdmi_mode=16

# Rotate LCD
lcd_rotate=2

Mirrors:

  • sudo nano /etc/apt/sources.list
# Zhejiang University Mirror
deb http://mirrors.zju.edu.cn/raspbian/raspbian/ jessie main contrib non-free rpi
deb-src http://mirrors.zju.edu.cn/raspbian/raspbian/ jessie main contrib non-free rpi

Shadowsocks Service

Json file

sudo nano /etc/shadowsocks/server.json

{
    "server":"0.0.0.0",
    "server_port":8388,
    "local_address":"127.0.0.1",
    "local_port":1080,
    "password":"your-passwd",
    "timeout":300,
    "method":"aes-256-cfb",
    "fast_open":✗,
    "workers":1
}

Service configure

sudo nano /etc/systemd/system/shadowsocks.service

[Unit]
Description=Shadowsocks

[Service]
TimeoutStartSec=0
ExecStart=/usr/local/bin/ssserver -c /etc/shadowsocks/server.json

[Install]
WantedBy=multi-user.target

systemctl enable shadowsocks
systemctl start shadowsocks
